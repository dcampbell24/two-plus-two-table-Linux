These are source files taken from XPokerEval and modified for easy compilation
of the TwoPlusTwo hand evaluator in a Linux environment. See LICENSES for
license information and links to the original sources.

Follow these steps in order to compile the TwoPlusTwo table:

1. Install Wine (http://www.winehq.org/)
2. Run the command 'c++ -I /usr/include/wine/windows/ *' in the src directory
   (adjusting the path if needed to point to your Wine header files).
3. Run './a.out'.
